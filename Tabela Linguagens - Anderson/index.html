Quadro Comparativo de Linguagens

Linguagem Vantagens Desvantagens Sua Opinião
Javascript Amplamente utilizado no desenvolvimento web Problemas de compatibilidade entre navegadores É a linguagem camaleão, serve para tudo.
Java Portabilidade e forte tipagem Sintaxe verbosa Ótima segurança, porém, sintaxe muito grande.
PHP Facilidade de integração com servidores web Problemas de segurança em versões mais antigas Ótima linguagem para aprender POO.
Python Sintaxe clara e legível Desempenho inferior em comparação com algumas linguagens A melhor linguagem, versátil, muito útil pra IA e ML.
C# Amplamente usado para desenvolvimento Windows Limitado a plataformas Microsoft Muito utilizada no mercado.
Minha linguagem é: Python Versátil, útil para tudo, ótima para Área de Dados. Fracamente tipada. Adoro python, é a linguagem onde me encontrei. Não há melhor, para mim
